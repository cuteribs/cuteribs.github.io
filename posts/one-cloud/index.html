也许算不上NAS, 但玩客云真真切切算得上私有云

可爱的排骨

评测/单品

发布于：2018年9月12日

大家好, 我是可爱的排骨 :)

一. 前言

玩客云是迅雷公司7102年推出的下载机产品, 应该算是下载宝和赚钱宝的集合体. 从玩客云的主页和广告来看, 玩客云是以下片功能为主, 还带HDMI输出看片功能. 而其水晶标志和链克会员又表达着满满的挖矿欲意.

玩客云主机有蓝 黑 橙三种颜色, 排骨收到的是蓝色.

四核ARM CPU, 1G 内存, 千兆网口, 双USB 2.0, HDMI输出. 这配置作为下载机性能应该是足够了, 300M以下家用宽带下载速度拉满不成问题. 怎么看都是某家的盒子产品…

二. 开箱

包装盒尺寸为 170 x 120 x 85 mm. 黑色简约纸盒.

包装分两层, 上层是玩客云主机, 下层是随机附带的电池和网线包装盒.

蓝色玩客云主机外壳为蓝色磨砂铝合金, 出厂套有透明膜, 膜上印有开机引导.

下层小盒子装着一根半米来长的扁平网线, 线芯很细, 外皮印着6类 30AWG. 这30AWG能不能满足6类线标准呢?

下层大盒装的是电源, 线长约一米, 是个很常见的宽电压输入100-240V 输出 12V1.5A 的电源, 输出接口也是常见的3.5DC. 可以跟很多盒子/路由器混用.

玩客云主机正面覆盖了一块亚克力, 内有迅雷水晶LED灯.

背面有3.5DC电源接口, RJ45带灯网口, 双USB2.0口, HDMI输出口和SD卡口. 另还有一个二维码用来给APP识别主机用.

主机底面印有产品名称/型号/制造商, 这个底面造型决定了玩客云这个主机不能稳实的竖立摆放!

玩客云主机尺寸为 140x90x40mm, 跟台式机3.5寸硬盘相似, 不内置硬盘也不能装入硬盘!

三. 配置和使用

插上电源, 网线和移动硬盘, 玩客云就可以开始使用了.

1. 手机APP

用手机扫描主机背面的二维就开始下载和安装玩客云手机APP了.

第一次需要注册玩客云账号并登录才能开始使用玩客云APP.

APP 首页显示分类显示硬盘内的资源, 分类机制就是照片/视频/音乐/文档四组文件扩展名白名单. 所有玩客云的设置项都在个人中心里.

  • 我的链克: 玩客奖励计划内容 (本次报告中不涉及).
  • 消息中心: 玩客云官方新闻.
  • 云添加列表: 下载任务列表 (外网->玩客云).
  • 传输列表: 上传任务列表 (内网->玩客云).
  • 自动备份: 开关手机照片/视频的自动备份功能 (手机->玩客云).
  • SD卡备份: 备份SD卡内容 (不支持exFAT, 对NTFS支持有问题, 支持FAT32).
  • 设备管理: 设置和查看玩客云主机信息.
  • APP设置: 设置手机APP功能及账号信息.
  • 帮助与反馈: 玩客云的详细功能介绍, 非常有用, 五星推荐浏览.

点击APP下方中间的 + 号, 可以将手机上的照片和视频上传到玩客云连接的硬盘里, 也可以给玩客云添加下载连接, 让玩客云把文件下载到硬盘中.

云添加的搜索功能就是百度搜索 (也可以设置成google等), 实际上就是嵌入了一个浏览器, 并随时监控可能的下载连接.

当你从手机秋秋或微信中复制了动作片下载连接并切换到玩客云APP时, APP也会提示自动添加到云添加列表.

玩客云与其它带下载功能的盒子/路由器不同, 它内置的下载功能享受迅雷加速服务. 也就是说用玩客云下载, 相当于你在电脑上用迅雷超级会员账号下载一样, 内置会员下载加速效果. 对非矿工用户来说, 这应该是买玩客云最主要的理由吧.

玩客云的玩客奖励计划即链克区块链, 也就是挖矿. 家里有矿的感觉真不错, 利用白天不在家的剩余宽带资源挖矿.

挖了矿能换钱吗? 不能! 只能换爱奇异等少数视频网站会员卡和迅雷会员卡, 还有一些基本没人会去在意的会员卡.

一经加入后无法退出是什么鬼? 开通后, 玩客云一联机就自动变矿机么? 想到这些排骨缩了….. 还是不要加入这个”迷之玩客奖励计划”先吧…

在设备管理>局域网访问设置中, 可以开启DLNA 服务, SAMBA 服务, HDMI 视频输出.

DLNA 已经过时了, 或者说在国内就根本没流行起来过.

SAMBA 就是平时我们用的SMB文件共享服务.

WIN10用户如果无法正确打开玩客云硬盘目录, 可以尝试在组策略设置中找到上图这个设置”启用不安全的来宾登录”. 排骨亲测可行, 而且不需要启用 SMB 1.0 支持.

HDMI 功能让玩客云可以直接将硬盘中的视频输出到电视/显示器上. 视频选择和播放控制需要用手机APP完成.

  1. 将电视的HDMI连接到玩客云背面, 用手机APP选择视频播放.
  2. 在APP右上角点击TV图标, 选择HDMI播放.
  3. 视频在电视上播放时, 可以通过手机APP控制进度. 注: 这种播放并不是平时使用的Airplay投屏, 视频内容并不是从手机发出, 而是玩客云直接从硬盘读取视频并解码再通过HDMI输出到电视上.

    玩客云也有Windows版 APP, 不过功能比较少. 没什么介绍和推荐的必要.

四. 功能与性能

1. 硬盘读写

玩客云帮助内容中说支持 NTFS/EXT4/HFS+ 三种硬盘格式, 排骨使用的是Windows用的NTFS格式.

排骨使用WIN10通过千兆网SMB服务将玩客云连接的移动硬盘映射后, 使用 CrystalDiskMark 测试读写速度, 连续读写都在40MB/s左右, 接近USB2.0上限速度, 完全满足家用环境的文件读写, 高速下载和高清视频播放要求. 如果玩客云配备的是USB3.0, 速度应该能更快.

2. 云下载性能

玩客云自带下载加速功能, 大部分资源都能满速下载, 省去了每年150元的迅雷会员费用. 某些被举报的资源也能下载, 只是没有了下载加速, 速度可能不美.

3. 手机/SD卡备份功能

玩客云支持将手机/SD卡上的图片/视频上传到所连接的硬盘中以达到备份的目的. 手机图片和视频是通过玩客云APP上传, 而SD卡是插入玩客云背面卡槽上传.

SD卡好像不支持exFAT格式, 排骨尝试使用NTFS格式时发现备份任务卡住了, 最后使用FAT32备份成功.

超过32G的SD/TF卡可能使用上图这个绿色工具格式化成FAT32格式.

4. 视频播放功能

玩客云不只是个下载机, 它与路由器挂硬盘的最大区别就是支持视频解码与HDMI输出. 也就是说如果你家需要的只是下载机+播放机, 也许跟群晖威联通相比玩客云是更适合的选择哦?

目前通过HDMI播放支持以下格式的视频： 3gp、flv、mkv、asf、mkv、mov、mp4、mpg、rmvb、vob、avi、rm；支持以下格式的音频：aac、mp3、aifc、wav、wma；

玩客云可以有以下几种视频播放模式:

HDMI直连电视解码播放, 省了个网络机顶盒.

NAS模式, 最常用也是最推荐的播放模式.

DLNA投屏模式, 有点绕但确实能这么用…

手机APP投屏模式, 效果较差, 不推荐.

5. 私有云功能

除去下载/备份/视频播放功能外, 玩客云还有一个重要的功能就是私有云. 类似群晖的QuickConnect, 用户可以使用手机APP通过迅雷的流量中转访问到家中的玩客云, 让没有公网IP或玩不转端口映射的用户也能享受私有云服务了.

迅雷为玩客云提供了最高 4MB/s 的中转速度 (相当于32Mbps的带宽, 完爆 QuickConnect. 电信200M宽带才给20M上行带宽), 这个服务是超值的! 这也让玩客云比其它产品和方案更有资格称作私有云.

买过马云马化藤云的同学应该很清楚一年10Mbps带宽的VPS搭个中转/穿透服务费用有多贵吧?

五. 总结

如果迅雷一直运营下去, 玩客云相关的服务能稳定的用上几年的话, 599元的玩客云的的确确是个非常值得买并且性价比也很高的产品, 绝对要比高价路由器挂硬盘好用实用耐用.

闲鱼转转走一波, 矿难玩客云老母鸡也是实惠的好选择哦.

优点

  1. 送下载加速服务, 省了会员年费.
  2. 送4MB/s外网中转服务, 省去了公网IP/端口映射之苦.
  3. 结合#2, 出差旅游备份照片非常便捷.
  4. 结合#2, 在外看片/给客户播放宣传片更轻松.

缺点

  1. 没有附送支架, 直立放置容易倒.
  2. 官网介绍太少, 很多比较重要和关键的信息都藏在FAQ里.
  3. 手机APP做得有些简陋, PC版APP简直鸡肋.
  4. 链克介绍不详细, 让人感觉一旦开通立马变肉鸡…
