黑苹果开荒记硬件篇: 新品 10700K+圣旗 Z490 配置实例

可爱的排骨

经验/装机

发布于：2020年7月10日

大家好, 我是可爱的排骨 😁

前言

618 刚过, 排骨一好友就扛来他新淘的主机, 找排骨给装黑苹果. 配的 CPU 主板内存套餐 (10700K + Z490 + 8G x 2), 显卡方面还在为买新的 RX 5600XT/5700XT 还是淘旧的 RX Vega 56/64 的问题上打架…排骨想着 Intel 台式机吃个黑苹果应该没什么难度吧, 网上随便找一找肯定有一堆华硕/华擎/技嘉/微星主板的现成 EFI 直接拿来用就是, 于是没多想就答应了.

可是骨感的现实啊…. 还好没翻车, 花了两周终于完成了 🤣

此开荒记是全新硬件从零开始吃黑苹果的记录, 没有任何现成的 EFI 可用. 由于篇幅较长, 排骨预计此开荒记至于会有两篇 😁

硬件配置

排骨收到配件大致如下: (价格是临时在京东淘宝上查的. 好像少了点什么? 🤨)

CPU: i7-10700K 作为 Intel 最新的酷睿十代高端 U, 拥有高达 3.8GHz 的基础频率和 5.1GHz 的最大频率. 而且它还能超频… 据说有人把基础频率超到了 5.3GHz! 目前 OpenCore 和

10700K 不超频是不是跟不带 K 的 10700 一样用? 当然不是! 两者对比请见上图. 10700K 基础频率要高出 900MHz, 也就是说带 K 的性能提升了 30% 以上 (理论上 😁).

另外 Intel 已经不给带 K 的盒装 CPU 送风扇了, 所以放心买散片吧!

主板: 圣旗 Z490-PRO GAMING 这款主板品牌排骨以前没接触到过, 听好友说是几年前华硕搞出来的子品牌 (类似以前的华擎). 那么设计生产用料和 BIOS 应该跟华硕是同一产品流水线出来了吧. 狮子头神兽 LOGO.

搞清楚了品牌的由来, 不再担心是广东山寨小厂出品就行了 (差点叫好友去退货…🤣). 华硕京东自营旗舰店就有卖圣旗主板, 无需担心贴牌和售后问题了.

装黑苹果本质就是给主板打补丁换驱动, 而给一款全新主板 (没有前人分享 EFI) 装黑苹果就相当于刚从黑石本蓝装毕业出来喊野队开荒 MC 啊…🤣 你得事先把主板的所有接口和位置都记清楚了再开始开荒哦.

圣旗 Z490-PRO GAMING 主板的配件不多, 一个背板, 几颗螺丝, 一张贴纸, 一张简易说明书再加一对 SATA 线就完事了.


圣旗 Z490-PRO GAMING 是一款基于 Intel Z490 芯片的 ATX 大板. 大板的好处就是给玩家提供最大的扩展空间 (6 条 PCIE 3.0 插槽): 一条主要的独显 x16 插槽, 一条副显卡用的 x16 (x16 尺寸, x4 效果) 插槽, 四条 x1 插槽.

PCIE 3.0 x1 最高带宽近 1GB/s, 可以安装 Wi-Fi 6 无线网卡, 万兆有线网卡, NVME 硬盘扩展卡, SATA 硬盘阵列卡等设备.

排骨不是电子电气相关专业人士, 关于供电用料方面的牛就不吹了. 这里排骨重点提一下开荒黑苹果时主板方便要注意的几点问题.

CPU 安装时务必注意两个缺口, 这一杆子压下去, 很可能就是上千元的损失啊… 即使自己装机经验再丰富也要小心上手! 😆

圣旗 Z490-PRO GAMING 的 M.2_1 口与 SATA6G_2 共用了一个 SATA 通道. 如果装了 M.2 接口 的 SATA 硬盘, 那 SATA6G_2 就失灵了.

圣旗 Z490-PRO GAMING 主板后置面板上有三组共六个 USB 口: 两个黑色 USB 2.0 口编号为 #9/#10; 两个蓝色 USB 3.0 口编号为 #5/#6; 两个 Tiffany 蓝的 USB 3.0 二代口编号为 #1/#2. (编号记下来, 后续要用到 😎)

圣旗 Z490-PRO GAMING 主板靠近 24pin 电源接口的位置有两个 USB 3.0 插槽 (USB32G1_34 和 USB32G1_78), 可以各自扩展出两个 USB 3.0 接口, 对应的是编号为 #3/#4 和 #7/#8. 这两个插槽一般用于外接机箱前置的 USB 面板上, 一般机箱带的面板只会带一个 USB 3.0 插头, #3/#4 和 #7/#8 两个插槽任选一个就行了. (编号记下来, 后续要用到 😎)

圣旗 Z490-PRO GAMING 主板四个 SATA 口旁边还有两个 USB2.0 插槽可扩展成四个 USB 2.0 接口. USB1112 对应编号 #11/#12, 而 USB_E12 对应的编号为 #13/#14. 机箱前置面板可能需要插一个, 另一个可以留给 PCIE 转接无线网卡上的蓝牙 芯片用. (编号记下来, 后续要用到 😎)

排骨将机箱的 USB 3.0 插头插在了 USB32G1_78 插槽了, 则机箱前端面板对应的两个 USB 3.0 接口对应如上图. 另外机箱的 USB 2.0 插头插在了 USB_E12 插槽上, 机箱面板只引出一个 USB 2.0 口, 编号是 #13. 目前 USB32G1_34 和 USB1112 插槽空置着, USB_E12 的 #14 口也空置了.

最后再提醒大家一下, 这款主板的板载网卡型号是螃蟹厂 Realtek 的 RTL8111, 板载声卡也是螃蟹厂的 ALC887, 都可以在黑苹果中完美驱动!

内存: 十铨 VULCAN TUF Gaming Alliance DDR4 2666 8G 是十铨与华硕 TUF Gmaing 品牌的联名电竞款内存. 这款黑配黄迷彩散热马甲的内存条标称频率是 2666MHz, 但插到圣旗 Z490-PRO GAMING 主板上默认运行频率只有 2400MHz. 因为支持 XMP2.0 一键超频, 所以排骨只需要在主板 BIOS 里开启动 XMP II 自动超频功能就能自动超频到 2666MHz 运行. 除了会提高一点用功耗外, 不会对 Windows 10 或 MacOS 有任何不良的影响.

双通道内存切记插 1/3, 不要插 1/2 槽, 不然没有双通道效果哦.

带马甲的条子就是为了超频散热准备的. 排骨十多年以来都用的 DDR2/DDR3 都是马甲条, 从金士顿的第一代 HyperX 蓝色马甲条开始, 放心超吧! (实测直接 BIOS 里把内存频率拉到与 10700K 标称的 2933MHz 完全没问题, 连电压都不需要手动加 😏)

机箱: 骨伽 (Cougar) 影武者X7 中塔. 这个家伙真是太大了… 排骨的小电脑桌下面的小庙根本容不下这个大佛🤣 只好扛到桌子上面来拍照. 左面是一整块侧透防爆玻璃面板, 这已经是时下机箱的标配了. 玻璃侧板四周均用金属边杠加固, 上下边也都加装了减震棉, 这一点设计可以极大的减小机箱的共振带来的噪音! 排骨的旧机箱经常嗡嗡嗡 要用双手压住左右两面的侧板才能安静下来😆 排骨深受其苦感慨良多哇

骨伽影武者X7 正面盖板靠左有一个竖直的凹槽, 是一条 RGB 灯条, 里面有一根小 3pin 的电源线.

骨伽影武者X7 顶板是大面积的散热孔, 可以安装 120mm/140mm 风扇, 足够安装 280mm/360mm 冷排.

顶板上还配了带磁吸带的防尘网面, 便于玩家拆装和清洁.

骨伽影武者X7 的前置面板也做到了顶板上, 提供了一个 USB 2.0, 两个 USB 3.0 数据接口, 一对耳机/麦克风接口, 电源/硬盘指示灯, 重启与开机按键. 鉴于这个机箱的高度…如果摆在电脑桌上享受光污, 那么开关机和拨插 U 盘就得要站起来了 🤣🤣🤣

骨伽影武者X7 长x高约为 45x49 厘米, 面积非常宽大. 从左面看, 可以装入比大板 ATX 规格更大的 EATX 主板.

正面和顶面均可安装 360mm 规格的冷排 (120mm x3).

背面仍有一处常规的 120mm 风扇位置.

底面电源位置上方还设计有两个 120mm 风扇安装位置.

骨伽影武者X7 右面主要用于走线和安装电源和硬盘.

底面空间除了下置电源位置了, 还内置了一个 3.5 寸硬盘架, 可以装入两块 3.5 寸标准硬盘.

中间附送了一块 RGB 灯光控制板, 提供了 8 个 3pin 的 RGB 灯光控制接口. 此灯控板附延长线可串联到华硕技嘉等主板的 12V/5V 光污控制接口上. 供电接口使用了一根 SATA 电源转接线 (考虑到目前的电源 SATA 多, 大 4pin 少). 另外灯控板还准备了一个 2pin 接口, 用于连接机箱的重启键, 让玩家直接用重启键控制光污效果~ 🙄 开荒黑苹果需要按无数次重启键, 还是老老实实把重启键接主板吧…

排骨建议骨伽直接在机箱上多加一个调光专用按键, 会更香~

电源: 骨伽 Cougar XTC 600W PFC 主动式电源. 转换率 83%, 80PLUS 白牌认证. 骨伽白牌 XTC 系列还有 400W/500W 规格的款式. 日系待机电容减少电流噪音的对耳朵的骚扰, 加上九页静音风扇使得这款电源非常的安静哦~ (实测声音很轻, 不是 YY 😎)

如果真要上 RX Vega 电烤箱的话, 还是升级到 GEX 金牌系列 750W 比较安心. 电脑电源这个东西, 对排骨来说就像保健品一样, 心理安慰远比实际效用重要. 😂

美洲狮神兽 LOGO.

骨伽 XTC 600W 提供一组 24pin 主板供电, 4+4pin CPU 供电 (支持), 两个 8pin PCIE 供电 (双显卡交火支持), 六个 SATA 供电 (硬盘和光污控制器会用到), 两个大 4pin PATA 供电 (除了风扇外, 基本用不到了). 可以涵盖几乎所有的使用情况了. (什么? 你要装 8 个硬盘?😲😲😲)


全模组电源: TT GT650W 全模组电源. 本来黑苹果装好准备让好友扛回去了, 结果他嫌骨伽这个非模组电源又黄又红的线不好看! 于是又拿来了一个 TT 饭盒让我帮他装上 🤣.

TT 这个 GT650W 的包装设计得太有爱了电源安装好饭盒不需要扔, 留着装多余的电脑材线, 螺丝等乱七八糟的小东西非常好用.

TT GT650W 达到 80PLUS 金牌认证标准, 相比白牌转化率要高7~8%, 当电脑实际功率是 200W 时, 用金牌电源耗电功率约为 222W, 用白牌电源约为 244W, 要多费 22W. 这多出来的 22W 功率就转化成热能等形式了, 功耗越大越明显. 所以换金牌的好处并不只是省电点, 同时还会降低机箱里的散热压力和噪音.

电源外壳四面都有散热孔, 七页的温控风扇也是非常安静的. 电容也是全日系的, 转化率更高, 发热更少, 更静音.

TT GT650W 采用全模组设计, 好处就是线材便于定制, 用多少插多少. CPU 与显卡都是 8pin 供电, TT 使用黑/红双色区别, 黑色为 CPU 供电, 红色为显卡供电.

附送的供电线有一根 20+4pin 主板供电线, 一根双头 4+4pin CPU 供电线, 两根双头 6+2pin 显卡供电线, 两根三头 15pin SATA 供电线, 一根四头大 4pin 供电线. 还有一根大小 4pin 转接线乱入.

目前这台配置除了必装的主板, CPU和显卡供电线外, 只需再插一根三头 SATA 供电线就够了, 一头接硬盘, 另一头给光污灯控器供电.

换上全模组的 TT GT650W 电源后, 整个机箱清爽了不少, 不再有花花绿绿的线缆横飞了. 只是附送的模组线长度和接头还不是最优解. 如果是排骨自己的电脑, 立马就把吃灰几年的镀银线和一堆接头搬出来自己做线了 😆 (送也送不出去好烦呀)

散热器: TT (ThermalTake) 冰龙240 RGB 神光同步水冷散热器. 都上 i7 了, 还带 K 了, 不上个水冷都不好意思么?

i7-10700K 这片 TDP 高达 125W 的 U, Intel 自己都不给配风扇了, 上个 240 水冷应该应该算是基础配置了, 除了散热比风冷好, 静音也是重要指标呀!

TT 的这款冰龙240 的 RGB 散热水泵模块通过两条纺织网笼 (旧款没有) 水管与 120x240mm 的冷排相联, 水泵引出两根排线, 一根是接主板的水泵接口或 CPU 风扇接口用于供电和调速的供电线, 另一根是接 RGB 灯控板用于接收光污显示指令的灯控线.

两个 120mm RGB 风扇需要自己安装在 240mm 冷排上, 与水泵一样他们也有一根供电线和一根灯控线. TT 冰龙240 配件里附带了小 4pin 二并一的延长线, 用于将两根风扇的供电线合并成一根插到主板的 CPU 风扇接口. 另外还附带了一根小 4pin 转大 4pin (PATA) 的线, 可于把风扇的供电改为电源直接供电, 这样可以减小主板用于压力 (不要小看风扇的功耗), 但是缺点是不能调速或监控转速, 风扇会从一开机就全速运转, 噪音最大化.

排骨非常相信圣旗 Z490-PRO GAMING 这块板子的供电能力, 不论是 11 相 DrMOS 还是 Digi+ VRM, 都是华硕中高端主板的加持技术. 把风扇供电交给主板说不定还能享受华硕的 AI Suite 3 的各种调速节能功能呢. (实测可以用 👻)

TT 冰龙240 还附带了一个光污控制器, 用来集合小 6pin 光污风扇的灯控线拼连接到主板的 RGB 灯控接口. 也可以使用控制器上的四个按键来手工控制风扇的转速和灯效.

硬盘: 东芝铠侠 TC10 480G. 一开始排骨就觉得好像少了点什么… 原来那猪友忘了给排骨硬盘了… 😱 没硬盘装什么鬼系统啊~ 网吧无盘站啊?

还好排骨上个月在 pdd 拼黑卡买了块 2.5 寸的铠侠 TC10 480GB SATA 固态硬盘, 装在绿联硬盘盒里当移动硬盘在用. 先将就一下用着吧, 不要耽误开荒进度了.

480GB 容量可以一分三区, MacOS, Windows 10 各用 160GB, 剩下的 160GB 格成 exFAT 作为数据存储和交换空间, 给 Mac/Win 共用. 完美!

当然最完美的方案排骨是推荐两条 NVME 硬盘分别安装 MacOS 和 Windows 10, 然后装一块 RAID 卡配两块 3.5 机械硬盘做 RAID 0 来放数据和交换.

显卡: XFX 讯景 RX 480 8G 小黑狼. 因为显卡基友还没买, 排骨只好把自己机器里矿卡先拿来顶上. 感觉散热不太好, 有空拆了用可悠然水洗一下, 再抹点芦荟硅藻泥. 🙃

键盘: 为了配合这个光污 TT 冰龙240水冷, 排骨把珍爱的杜伽 K520 拿出来拍照用. 目前入手最漂亮的光污键盘, 下次再单晒. 😍

鼠标: 十几块钱的英菲克鼠标, 一言难尽, 已经退货了…🤣

成效

整机安装完成后就是这个样子, 走线非常清爽. 8pin 的 PCIE 黄色供电线有点搅局 🤣 整机待机声音非常轻, 耳机凑近了也听不到什么声音.


配上排骨的杜伽 K520 光污键盘和 AOC 4K 显示器配小米显示器挂灯, 有一点点骚气的紧凑型黑苹果电脑生产/娱乐环境就达成了.

圣旗 Z490-PRO GAMING 主板的 BIOS 对华硕主板用户来说是不是非常熟悉? 除了左上角缺失的 ASUS LOGO 外基本就是一模一样! 说明圣旗主板毫无疑问是真正的华硕血脉.

水泵速度始终以3200多转的全速运行, 但是排骨听不到一丝噪音. 得益于圣旗 Z490-PRO GAMING 主板 (或者说是华硕先进 BIOS) 的 QFan 智能风扇调控, 系统空载时冷排上的两个 120mm 风扇只保持了 590 rpm 的低转速运行, 当 CPU 负载上升时, 风扇转换也是跟着提高到 1800 rpm 的峰值.

圣旗 Z490-PRO GAMING 主板与十铨 VULCAN TUF Gaming Alliance 同时支持 XMP 2.0 AI 超频调控, 排骨只需要把内存频率直接拉到高 (2933MHz), 主板就自动将内存电压从从默认的 1.20V 调升至 1.35 V 左右.

开荒完成的黑苹果系统为最新的 10.15.5 Catalina, 11.0 Big Sur 还只有开发版, 吃起来难度还很高.

可以正确识别 CPU 档次, 频率与核心数.

内存容量与运行频率也准确识别出来.

独立显卡也能原生正确驱动.

i7-10700K 与圣旗 Z490-PRO GAMING 同样可以享受 MacOS 系统的原生电源管理, 无需第三方驱动或工具就能体验到与白苹果相同的 CPU 变频体验. 自动睡眠与唤醒功能都可正确使用, 不会陷入一睡不起的尴尬境地.

没有苹果原生电源管理的主板无法正确自动调节 CPU 的运行频率, 会使 CPU 莫名兴奋而导致功耗变高, 温度上升, 散热风扇长时间保持高速运转. 又热又吵.

讯景 RX488 矿卡原生完美驱动运行, 接上 AOC 4K 显示器直接实现 HiDPI, 无需打补丁装驱动注入 EDID. 4K 双显应该也不成问题, 下次入手一台明基 4K 来体验一下.

圣旗 Z490-PRO GAMING 板载声卡与 RX480 DP 音频输出都能正常识别和使用.

RTL8111 有线千兆网卡正常运行, 完美驱动无难度,

RTL8125 有线 2.5 千兆也能正常驱动, 只不过需要手工设置协议速率 (1000M 或 2500M), 目前驱动还不能自动协商速率. (下篇加入)

双系统三分区使用正常. 无名的 NTFS 分区装着 Windows 10, 开机时可以直接从 OpenCore 菜单选择进 Win10 还是 MacOS.

ExFat 分区可以分别从 Win10 和 MacOS 读写访问, 不像 FAT32 那样有单文件最大 4GB 的瓶颈, 容量也高得多, 格外便利.

USB 控制器全部端口都开启, 包含 14 个 USB 2.0 和 8 个 USB 3.0.

Geekbench 5 单核跑分 1300, 多核 8700, 与其它主板 + 10700K 的分数在一个水平线了. 说明这次黑苹果开荒不存在 CPU 方面的性能问题.

顺便也把矿卡跑了一下分, 也在 RX480 的正常水平, 代表原生驱动很完美.

小结

排骨此次黑苹果开荒前后一共用掉近两周时间, 通过学习和使用第一次接触的 OpenCore 启动加载系统将一款全新的主板圣旗 Z490-PRO GAMING 主板带入了黑苹果的世界. 这款主板对黑苹果系统的兼容性还是很高的, 其它方面坑点主要就是:

  1. OpenCore 比较新, 参考资料几乎只有官方推荐的新手教程, 遇到问题能搜到的条目也很少. 🙄
  2. Z490 和 10700K 使用的 Comet Lake (彗星湖) 架构也太新, 很多前人分享的 1xx/2xx/3xx 配置方法并不适用. 造成了很多坑.
  3. OpenCore 的配置工具 OpenCore Configurator 经常把我的配置文件清空~ 😣
  4. 加入 Z490 果友秋秋群太晚~

目前成功启用的功能:

  1. CPU 变频工作正常
  2. 独显 AMD RX480 工作正常.
  3. 板载 Realtek 有线网卡工作正常.
  4. 板载 ALC887 集成声卡工作正常.
  5. 睡眠唤醒工作正常.
  6. 4K 显示器 HIDPI 自动开启, 支持夜览模式.
  7. 所有 USB 接口工作正常, 完美定制.

待解决的问题:

  1. 核显 Intel UHD 630 虽可独立工作, DP 与 HDMI 口都能输出, 但是无缓存, 显示很卡.
  2. 插入独显后, 核显设备被主板自动屏蔽… 在 Win10 和 MacOS 中都找不到设备, 自然也无法辅助解码.

后续要进行的更新:

  1. 驱动 DW1560 无线网卡. 这个卡可原生驱动, 已在排骨的 XPS 9360 黑苹果里服役了两年.
  2. 驱动 AC9260 无线网卡. 此卡目前已经能驱动, 但驱动程序问题很多, 速度也很慢.
  3. 驱动 RAID 卡, 同时支持 Win10 和 MacOS. Z490 主板虽然板载RAID 功能, 但 MacOS 不认, 必须使用独立的 RAID 卡才能实现双系统都能访问.
