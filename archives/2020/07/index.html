夜间生产力的辅助神器: 米家显示器挂灯

可爱的排骨

评测/单品

发布于：2020年7月21日

大家好, 我是可爱的排骨 😎

不知道大家晚上用电脑时有没有开灯的习惯? 不开灯会不会觉得屏幕太刺眼, 感觉眼睛不舒适?

排骨看电视或者用电脑时是一定要开灯的. 😎 灯光还不能太弱, 还得是白色光源, 这样眼睛才不会觉得累. 排骨不懂如何从光学与生理学角度来解释原因, 单凭日均面对显示屏 14 小时并持续了 20 余年的使用经验来说, 灯是一定要开滴! 🙃

受限于住房条件和使用便利问题, 很多家庭的电脑都安排在卧室里, 喜欢深夜开团的玩家或创作的达人, 开房间灯非常影响家人休息. 而用台灯又窘于桌面的空间和光照的角度. 需求创造市场, 显示器挂灯出现了~

排骨观望了某基上千元的显示器挂灯一年多, 小米携 Yeelight 横空强势推出了首款米家显示器挂灯, 首发众筹价只要 169 元.

黑苹果开荒记系统篇: 超详细的纯净 MacOS 安装流程

可爱的排骨

经验/装机

发布于：2020年7月10日

大家好, 我是可爱的排骨 😁

前言

这一篇排骨来说说 Z490 主板的黑苹果系统安装流程 (基于 OpenCore). 仍然是以上一篇的硬件 Intel i7-10700K + 圣旗 Z490-PRO Gaming + 讯景 RX 480 为基础, 介绍排骨开荒吃土的全部过程~ 🤣

流程大概有以下几步:

  1. 制作安装盘.
  2. 安装 Windows 10.
  3. 下载相关文件和脚本工具.
  4. 提取 DSDT (系统信息)并制作补丁 (SSDT).
  5. 创建 config.plist (OC 配置) 并复制到安装盘.
  6. 重新分区
  7. 安装 MacOS 10.15.5.
  8. 补全驱动并修正 OC 配置.
  9. 安装 Windows 10. (可选)

文本安装配置流程主要参考自 [OpenCore Desktop Guide].
由于平台限制要求, 本文所有需要下载的文件和程序都不能提供下载连接, 请充分自我发挥搜索本领~ 实在找不到可在评论区留言向排骨索取. 😅

可爱的排骨

经验/装机

发布于：2020年7月10日

大家好, 我是可爱的排骨 😁

前言

618 刚过, 排骨一好友就扛来他新淘的主机, 找排骨给装黑苹果. 配的 CPU 主板内存套餐 (10700K + Z490 + 8G x 2), 显卡方面还在为买新的 RX 5600XT/5700XT 还是淘旧的 RX Vega 56/64 的问题上打架…排骨想着 Intel 台式机吃个黑苹果应该没什么难度吧, 网上随便找一找肯定有一堆华硕/华擎/技嘉/微星主板的现成 EFI 直接拿来用就是, 于是没多想就答应了.

可是骨感的现实啊…. 还好没翻车, 花了两周终于完成了 🤣

此开荒记是全新硬件从零开始吃黑苹果的记录, 没有任何现成的 EFI 可用. 由于篇幅较长, 排骨预计此开荒记至于会有两篇 😁
